# Development Environment

## 導入

このドキュメントは、`alphaprobe` プロジェクトの開発環境について説明します。
このリポジトリで開発を始めるために必要なツールや、環境構築の背景にある思想を理解するための手引きです。

## 概要

すべての言語は "Single Folder Workspace" のコンセプトで設定されています。

## 開発環境に含まれる主要なコンポーネント

以下に、この開発環境に含まれる主要なツールと言語を示します。

### 対応言語

- Go
- Python
- Julia
- JavaScript & TypeScript

### 主要ツール

- **シェル**: `fish` がデフォルトシェルとして設定されており、補完機能などが強化されています。
- **バージョン管理**: `mise` を使用して、各種言語のランタイムバージョンをリポジトリ単位で管理しています。
- **Jupyter**: データ分析やプロトタイピングのためのインタラクティブな環境が利用可能です。
- **AIアシスタント**: Gemini CLI が統合されており、開発をサポートします。
- **Linter & Formatter**: 現状の最新のをプリインストールしてます。(`biome` や `black-formatter`など)
- **大規模ファイル管理**: `git lfs` が設定されており、大きなバイナリファイルなどを扱うことができます。
